# Getting Started Application

### Clone the application

```console
$ git clone git@github.com:luaysuarna/quiz-master.git
```

### Setup Database

```console
$ rake db:migrate
```

### Run Server

```console
$ rails s
```

# Run Unit Test

run this command from your root rails application

```console
$ rspec
```

#### nb: We need firefox version 27.0 for test features
